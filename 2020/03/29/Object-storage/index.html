8个免费对象存储空间云存储服务对比及选择建议

目前我们建站项目或者是存储需求使用较多的偏向于云服务器、对象存储、云数据库，当然如果我们复杂工艺项目可能还会有其他云产品组合需求。对于我们大部分网站项目来说，或者是自建网盘图床用户需求，对象存储/云存储使用的还是比较多的，我们可以通过将较大的图片、静态文件分离至第三方对象存储空间，来降低云服务器本身的带宽占用问题，进而可以提高网站的访问速度。

第一、亚马逊提供5GB对象存储

我们还是比较熟悉的亚马逊云服务AWS，有提供新注册用户免费12个月的云服务器，同时也有提供5GB对象存储12个月的服务，如果我们有需要的可以根据需要选择。

这我们可以看到有提供Amazon S3对象存储服务。5GB存储空间对于小型项目来说是够用的。

第二、网易云对象存储免费50GB

网易云也有提供云计算服务，但是这个商家偏向于企业用户，需要企业认证资料提交之后然后可以免费对象存储有50GB存储空间，20GB下行流量，每月100 万次/月 GET 请求。

我们个人用户可能是无法申请的，这个商家偏向于企业用户需求。

第三、腾讯云新用户免费50GB存储

腾讯云新注册用户首次使用COS对象存储和CDN内容分发的时候会告知我们直接送6个月的50GB存储和50GB每月流量包。但是只赠送六个月，如果后面我们需要使用会根据使用量计费，我们可以单独购买流量包或者存储包。腾讯云对象存储的好处就是支持提供免费域名，我们可以不绑定域名直接使用他们提供的。

第四、UCLOUD免费20GB存储和流量

UCLOUD商家原来是主做企业用户的，对于个人用户群当然是可以购买只不过预算是比较大的。最近两年也开始慢慢下探个人用户市场，有提供代金券领取后体验云服务器等资源但是续费还是比较贵的。不过UCLOUD商家有提供免费20GB对象存储和每月20GB外网流量。

第五、七牛免费10GB存储空间

七牛云是我们比较熟悉的老牌云存储服务商，这几年虽然在云服务器上没有抓住机会，但是在云存储上面的优势还是明显的。目前七牛云有提供 会员免费10GB存储，但是不支持HTTPS，如果需要用到HTTPS则需要单独计费的，如果我们邀请更多的用户加入七牛云还可以得到更多的存储空间。

第六、又拍云免费10GB存储空间

加入又拍云联盟的站长，需要将网站底部加上又拍云的LOGO，申请加入联盟，可以得到免费10GB存储空间和15GB每月流量，对于一般的网站是足够用的。而且还支持HTTPS，这一点上比七牛云存储会好一些。唯一不足的就是需要我们加上LOGO，如果被发现取消则会被取消资格，第二年依旧是免费自动续签。

第七、青云免费10GB存储空间

青云（QingCloud）是国内的一家云产品提供商之一，看到他们也有提供免费对象存储。提供10GB存储空间。

第八、景安网络免费5GB对象存储

景安网络国内的一家云服务企业在众多云商家大佬夹缝中生存还是不容易的。早年这个服务商以其独特的服务吸引用户，但是如今优势也逐渐减少，竞争压力也是比较大的。商家也有提供5GB对象存储，但是并不是直接免费，用户需要付费一并购买才可以包含在免费额度中。
